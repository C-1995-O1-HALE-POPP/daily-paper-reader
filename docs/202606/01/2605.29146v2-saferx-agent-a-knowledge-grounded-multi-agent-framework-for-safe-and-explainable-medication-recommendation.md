## 摘要
药物推荐预测患者就诊时的用药方案，但现有方法仍面临两个关键挑战。在模型层面，传统药物推荐方法仅预测结构化药物代码，且可依据的证据有限，而大语言模型智能体虽能利用更丰富的临床上下文，但可能缺乏安全性验证和可追溯性。在任务层面，现有基准常使用宽泛的药物类别，忽略了亚组级别的安全性差异，可能导致风险高估。我们引入了首个基于四级ATC代码生成的细粒度药物推荐设置。我们提出了安全处方智能体（SafeRx-Agent），这是一个基于知识的多智能体框架，利用患者上下文、外部临床知识和安全性验证来推荐可追溯的药物组合。在MIMIC-III和MIMIC-IV数据集上的实验结果表明，SafeRx-Agent在控制药物相互作用、禁忌症和药物组合规模的同时，提高了细粒度药物预测的准确性。

## Abstract
Medication recommendation predicts medications for patient visits, but existing methods still face two key challenges. At the model level, traditional drug recommendation methods only predict structured drug codes with limited evidence grounding, while LLM agents can use richer clinical context but may lack safety verification and traceability. At the task level, existing benchmarks often use broad medication categories, which ignore subgroup-level safety differences and can lead to risk overestimation. We introduce the first fine-grained medication recommendation setting based on fourth-level ATC code generation. We propose Safe Prescription Agent (SafeRx-Agent), a knowledge-grounded multi-agent framework that uses patient context, external clinical knowledge, and safety verification to recommend traceable medication sets. Experimental results on MIMIC-III and MIMIC-IV datasets show that SafeRx-Agent improves fine-grained medication prediction accuracy while controlling drug interactions, contraindications, and medication set size.

## 2. 方法论

### 核心思想
- **SafeRx-Agent**：基于知识的多智能体框架，通过将患者病例路由至多个专科专家智能体，生成ATC-L4候选药物，再经全局评论和安全验证产生最终可追溯的处方。

### 关键技术细节（算法流程）
1. **专家面板构建**：聚类患者嵌入得到7个集群，映射至ICD章节，构造7个专科专家+1个始终激活的通用支持专家。
2. **路由（ROUTE）**：根据当前和历史ICD代码与专家范围的匹配得分，稀疏激活相关专家（通用专家始终激活）。
3. **专家摘要（SUMMARIZE）**：每个激活专家提取与自身专科相关的患者信息，生成结构化摘要和过往就诊解释。
4. **药物生成（GENERATE）**：每个专家基于摘要、MEDI指示资源、患者记录，生成排序的ATC-L4候选药物列表。
5. **全局评论（CRITIQUE）**：合并所有专家的候选集，检查重复、缺乏证据的药物，执行保守修剪。
6. **安全验证**：
   - **FIND FLAGS**：使用DDI矩阵（TWOSIDES）和禁忌症矩阵（openFDA）扫描候选集，标记风险项。
   - **VERIFY**：对每个标记项，结合患者上下文（如是否既往用药、诊断）决定保留或移除，生成可追溯报告。

### 关键公式/矩阵
- 使用对称的DDI矩阵 \( M_{DDI} \in [0,1]^{|V_{med}| \times |V_{med}|} \)，禁忌症矩阵 \( M_{Contra} \in [0,1]^{|V_{med}| \times |V_{diag}|} \)。

## 6. 主要结论与发现

- **预测精度**：SafeRx-Agent在所有backbone上均优于直接提示和通用智能体基线，在MIMIC-IV上最佳F1达0.4325（Claude）和0.4138（GPT-5.2无安全过滤），且与监督学习基线相当或更优。

2. **多智能体协作+安全验证**：结合专科专家、全局评论、外部知识资源和规则/LLM双重验证，实现可追溯、可解释的处方。
